## The simples FreeRTOS project with LED blinking

Project includes FreeRTOS operating system and performs LED blinking
from task.

## Features
 - Clock source: HSE
 - SYSCLK: 480 MHz
 - APB1: 120 MHz
 - APB2: 120 MHz
 - CMSIS only
 - FreeRTOS

## Versions
  - IAR EWARM: v8.20
  - CMSIS: 4.5.0
  - FreeRTOS v10.2.1
